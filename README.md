Dot duration: 1 unit of time
Dash duration: 3 units of time
Gap between dots and dashes within a character: 1 unit
Gap between characters: 3 units
Gap between words: 7 units
These timing rules ensure that sequences like · · · are recognized as a single letter (S) rather than three separate letters (E E E)
